## Getting Started

To install and run the ai-go tool:
1. Clone the repository
   ```bash
   git clone https://github.com/your_username/ai-go.git
   ```
2. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```
3. Run the tool
   ```bash
   python main.py
   ```
